For this project, we focused primarily on functionality and usability rather than visual styling. The design is intentionally minimal and straightforward, allowing users to easily interact with the Kanban-style boards and tasks without distraction.

Overall Style & Theme: Basic and clean, with a focus on clarity and responsiveness of task and board management.
Color Palette: Default browser styling; no specific colour scheme was applied.
Design Inspiration: The project was inspired by general Kanban boards commonly used in productivity tools, but no specific website or template was copied.

The main priority was implementing all required features manually using React and native browser APIs, ensuring correct state management, task creation/editing, drag-and-drop functionality, and persistent data storage via localStorage. Functionality and smooth user interaction guided the design decisions rather than aesthetics.
